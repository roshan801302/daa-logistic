📊 Algorithm Comparison
AlgorithmDistanceRuntimeQuality
🔷 Dijkstra
O((V+E) log V)
Classic greedy shortest-path. Uses a priority queue to relax edges. Guarantees optimal paths on non-negative weighted graphs.
⚡ CH
O(n log n) query
Contraction Hierarchies. Preprocesses graph by contracting low-importance nodes and adding shortcut edges. Bidirectional Dijkstra on the hierarchy.
🧩 CRP
O(B²) customization
Customizable Route Planning. Partitions the graph into cells/regions. Precomputes boundary node paths; customization adapts to changing edge weights.
🚫 Tabu Search
O(n² × iter)
Metaheuristic for TSP/VRP. Uses 2-opt swaps to improve tour. A tabu list prevents revisiting recent moves, escaping local optima.
🐜 ACO
O(ants × n² × iter)
Ant Colony Optimization. Multiple ants construct tours guided by pheromone trails. Good tours reinforce pheromones; trails evaporate over time.
